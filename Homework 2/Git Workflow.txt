Git-ի աշխատանքային պրոցեսը(workflow) բաժանված է երեք վիճակի.

Աշխատանքային կատալոգը - Փոփոխեք ֆայլերը ձեր աշխատանքային կատալոգում
միջանկյալ տարածք (Ինդեքս) - նախապատրաստեք ֆայլերը և ավելացրեք դրանց նկարները ձեր միջանկյալ տարածքում
Git կատալոգ (պահեստ) - Կատարեք մի հանձնում, որը մշտապես պահում է լուսանկարները ձեր Git կատալոգում: Ստուգեք առկա ցանկացած տարբերակ, կատարեք փոփոխություններ, ինդեքսավորեք դրանք և կատարեք
