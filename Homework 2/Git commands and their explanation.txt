Ստեղծում է նոր (Repositorie)
      git status
ցուցադրում է ինդեքսավորված ֆայլերի աշխատանքային կատալոգի և բաժանման կարգավիճակը :
      git push
ուղարկում է նշված ֆայլերը նշված (Repositorie)
      git pull
Git pull հրամանը օգտագործվում է հեռավոր պահոցից բովանդակություն քաշելու և ներբեռնելու համար և անմիջապես այդ բովանդակությամբ տեղական պահեստը թարմացնելու համար :
      git add
Git add հրամանը փոփոխություն է ավելացնում աշխատանքային կատալոգից  ինդեքսավորված ֆայլերի բաժին :
      git commit
Git commit հրամանը նկարում է նախագծի համար ինդեքսավորված ֆայլերի բաժնում ավելացված փոփոխությունների ներկա վիճակի պատկերը :
